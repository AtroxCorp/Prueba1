<h1>COMANDOS BASICOS DE GIT</h1>

git init - agrega las carpetas dentro del sitio de trabajo para administrar por git

git add - se añaden archivos en git

git status - muestra el estatus actual de los archivos dentro de git

git commit -m "nameversio" - basicamente un punto dde guardado de la version a modificar

git config --global user.email "emailuser@emial.co" - configuracion del correo del programador

git config --global user.name "nameuser" - configuracion del nombre del programdor

git log - para ver todos los commits y sus respectivos cambios

git checkout - para revertir los cambios hasta el ultimo punto de gurdadoen git

git diff - para ver las diferencias si los arcchivos fueron modificados

Nota: Las carpetas se escriben solas y los archivos con su terminación.
